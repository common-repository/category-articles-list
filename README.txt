List all the posts in a given category.

== Description ==

The Category Articles List plugin lists all the posts in a given category.

== Installation ==

1. Upload `category-articles-list` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
